# 🚪 智能寝室门禁报警器 (Smart Dorm Alarm)

## 📌 项目简介
基于 STM32F103C8T6 的简易安防报警系统。当检测到寝室门异常开启时，自动触发蜂鸣器报警。
这是我（大二上）的寒假练手项目。

## 🛠️ 硬件清单
- **主控**：STM32F103C8T6 (Blue Pill)
- **传感器**：干簧管门磁模块 (常闭型)
- **报警器**：有源蜂鸣器 (低电平触发)
- **电源**：9V 电池 + DC 转接头

## ⚙️ 功能特性 (V1.0)
- [x] 门开检测（GPIO 输入）
- [x] 报警触发（蜂鸣器驱动）
- [x] 软件防抖 (20ms)
- [ ] 待开发：布防/撤防模式 (V2.0)

## 📸 接线说明

| :--- | :--- |
